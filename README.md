## ğŸš€ Getting Started

Project ko apne computer par chalane ke liye neeche diye gaye steps follow karein.

### Prerequisites (Zaroori Cheezein)
*   Node.js (v14+)
*   MongoDB (Local ya Atlas URL)

### Installation

1.  **Repo Clone karein:**
    ```bash
    git clone https://github.com/your-username/your-repo-name.git
    cd your-repo-name
    ```

2.  **Backend Setup:**
    ```bash
    cd backend
    npm install
    # .env file banayein aur usme DB URL aur PORT add karein
    npm run dev
    ```

3.  **Frontend Setup:**
    ```bash
    cd ../frontend
    npm install
    npm start
    ```
